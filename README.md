# dotfiles

Dot Files for my Linux Configurations

## Location of Dot Files:

### 1. starship:

>     ~/.config/starship.toml

### 2. neofetch:

>     ~/.config/neofetch/config.conf

### 3. bash:

>     ~/.bashrc

### 4. fish:

>     ~/.config/fish/config.fish

### 5. completions(exa):

bash:
>     /etc/bash_completion.d/

fish:
>     ~/.config/fish/completions
